# BACKBURN

## A Ludum Dare #32 Game

### Developing

```shell
npm install
npm start # Starts a live-reload webserver on http://0.0.0.0:3000
```

### Building/Deploying

```shell
npm install
npm run build # Creates files dist/bundle.js
# Deploy bundle.js, index.html, and img/* to some static hosting
```
